Getting Started

Prerequisites:

  • API Key — provided by Enervibe.
  • Query ID — the predefined query you intend to run.
  • Network access — outbound HTTPS to the API.

API URL

https://ev-api.valiuse.com

Basic Usage

Health Check

https://ev-api.valiuse.com/healthz

Expected: {"ok": true}

Run a Query

https://ev-api.valiuse.com/run?key=YOUR_API_KEY&q=YOUR_QUERY_ID

Replace YOUR_API_KEY and YOUR_QUERY_ID with real values.

Available Data Types

1) Vehicles Weight

Query ID: 5EF690DC-092B-4176-A7B4-16408FAF0B9E

  • Description: Returns a list of vehicles and their latest weight measured within the last X minutes (default 15).
  • Use cases: Load management, compliance, diagnostics.

Parameters

NameTypeNotes
minutesintegerDefault 15; range 1–500.

2) Wheels Telemetry

Query ID: 8394F36D-2C9C-4871-AB8A-5489175E32E4

  • Description: Returns a list of telemetry measurements for all wheels of all vehicles within the last X minutes.
  • Use cases: Maintenance, early detection, safety.

Parameters

NameTypeNotes
minutesintegerDefault 15; range 1–500.

3) Active Alerts

Query ID: 47CE71C4-7406-4207-A0E9-ACBA0382CC18

  • Description: Recent alerts (flat tire, distance threshold, imbalance, overload).
  • Use cases: Real-time monitoring and operations.

Parameters

NameTypeNotes
hoursintegerDefault 168; range 1–2200.
include_closed0/1Include resolved alerts; default 0 (no).

Examples

Vehicles Weight (last 30 minutes)

https://ev-api.valiuse.com/run?key=YOUR_API_KEY&q=5EF690DC-092B-4176-A7B4-16408FAF0B9E&minutes=30

Wheels Telemetry (15 minutes) as CSV

https://ev-api.valiuse.com/run?key=YOUR_API_KEY&q=8394F36D-2C9C-4871-AB8A-5489175E32E4&format=csv

Alerts (last 24 hours; include resolved)

https://ev-api.valiuse.com/run?key=YOUR_API_KEY&q=47CE71C4-7406-4207-A0E9-ACBA0382CC18&hours=24&include_closed=1

Response Options

  • JSON (default).
  • CSV — append &format=csv.
  • Demo mode — append &demo=true to return sample data. Requires a valid access key and Query ID (q).

Using in Tools

Excel / Google Sheets

  1. Open Excel or Google Sheets.
  2. Use “Get Data from Web” / “Import”.
  3. Paste your API URL with &format=csv.

Web Browser

Paste the URL into the address bar to preview JSON/CSV.

Troubleshooting

IssueResolution
“Invalid API key”Verify value and expiration.
“Query not found”Confirm Query ID and permissions.
“Service unavailable”Retry later; contact support if persistent.
No data returnedTest &demo=true; adjust lookback (minutes/hours).
